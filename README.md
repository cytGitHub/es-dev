# es-dev-simple

## 特点
* 不需要配置`webpack`，直接上手
* 支持`react`，`antd`

## 使用
* `npm install es-dev-simple -g` 全局安装
    ```
    创建目录
    test-mkdir/src/index.js
    ```
* 在`test-mkdir`目录执行`es-dev` 启动开发服务

* 所有命令
  * `es dev` 快速启动服务
  * `es build`
    * `-c`，`--component` 构建`react`组件
    * `-e`，`ES6`构建到`ES5`
